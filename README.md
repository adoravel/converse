# converse

A minimal, client-side Minecraft mod focused on improving the chat experience

## Features

- Hide and show the chat with a keybind (period (`.`) by default)
- Smart chat background width (dynamically shrinks the chat background to fit the actual width of each message)
- Hides the ~~evil~~ colored square indicator next to chat messages
- Smoothly animate new chat messages and the chat input field opening
